# RetailOps Insight

RetailOps Insight is a practical data analytics, reporting, data quality, and application support portfolio project.

## Project Goal

This project analyzes real retail sales data and combines it with support-style issue investigation to demonstrate practical skills for junior data, BI, reporting, operations, data quality, and application support roles.

## Tools Planned

- Python
- Pandas
- SQL
- SQLite
- DBeaver
- Jupyter Notebook
- pytest
- Great Expectations
- GitHub Actions
- GitHub Issues
- Docker
- Metabase

## Dataset

Main sales dataset: UCI Online Retail dataset.

Additional support-ticket and POS issue data will be created synthetically to simulate real application support and reporting problems.
