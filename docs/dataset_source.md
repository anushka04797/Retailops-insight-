# Dataset Source

## Main Dataset: Online Retail

Source: UCI Machine Learning Repository  
Dataset name: Online Retail  
Dataset type: Transactional retail sales data  

## Description

This dataset contains transactional records for a UK-based, registered, non-store online retail business. The business mainly sells giftware, and many customers are wholesalers.

## Date Range

2010-12-01 to 2011-12-09

## Why this dataset was selected

This dataset was selected because it represents real retail transaction data and is suitable for demonstrating:

- sales analysis
- product/customer analysis
- data cleaning
- missing value handling
- cancelled/refunded transaction investigation
- SQL analysis
- KPI reporting
- dashboard creation
- support-style reporting issue investigation

## Planned Project Use

The raw dataset will be preserved in `data/raw/`.

Cleaned and transformed datasets will be saved in `data/cleaned/`.

A SQLite database will be created in `data/database/`.

Synthetic support tickets and POS/reporting issue logs may be created later to simulate application support and reporting defect investigation.
